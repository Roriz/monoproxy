## Design Context

### Users
- **Target Audience**: Readers of web novels (specifically "Lord of the Mysteries").
- **Context**: Deep, immersive reading and analysis of complex fantasy fiction.
- **Job to be Done**: Explore character relationships, data visualizations, and story analysis in a premium, editorial environment.

### Brand Personality
- **Voice**: Authoritative yet accessible, academic but engaging.
- **Personality**: Editorial, precise, immersive.
- **Emotional Goals**: Wonder, clarity, awe.

### Aesthetic Direction
- **Visual Tone**: Editorial Magazine / Dark Scholarly.
- **References**: High-end digital longform journalism (e.g., NYT Snowfall, ProPublica).
- **Theme**: Dark mode prioritized (for overnight reading), with high-quality typography.

### Design Principles
1. **Vertical Rhythm**: Every element must respect the modular type scale and spacing system.
2. **Data Brilliance**: Data visualizations (funnels, donuts) must feel alive and perfectly integrated, not like widgets.
3. **No AI Tells**: Zero side-stripe borders, zero generic gray text on dark backgrounds.
4. **Purposeful Motion**: Subtle, staggered reveals to reinforce the "mystery" and "discovery" themes.
